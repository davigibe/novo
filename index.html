Introdução ao HTML e CSS

Tags HTML

A tag <p> cria um parágrafo em HTML, sendo uma das tags mais básicas para estruturação de texto.

As tags <h1> até <h6> definem cabeçalhos, variando do maior (<h1>) ao menor (<h6>).

Para inserir uma imagem, utiliza-se a tag <img>, que necessita do atributo 'src' para especificar a fonte da imagem.

Estilizações CSS

A cor do texto é definida com a propriedade 'color' em CSS, podendo ser especificada de várias formas, como nomes de cores ou códigos hexadecimais.

Centralizar elementos pode ser feito com 'text-align: center' para textos e 'margin: auto' para elementos de bloco.

O tamanho da fonte é controlado com a propriedade 'font-size', podendo ser definida em diversas unidades como pixels, em, rem, etc.
